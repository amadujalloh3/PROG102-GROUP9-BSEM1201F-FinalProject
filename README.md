#  AgriPrice Alert System

**A Digital Public Good for Agricultural Market Price Alerts**

PROG102 - Principles of Software Engineering | Final Project
BSEM1201F | Group 9

##  Overview

AgriPrice Alert System is a web-based Digital Public Good designed to empower 
smallholder farmers in Sierra Leone with real-time agricultural commodity price 
information. The system helps farmers make informed decisions about when and 
where to sell their produce, reducing exploitation by middlemen.

##  SDG Alignment

This project aligns with **UN Sustainable Development Goal 2: Zero Hunger**
- Target 2.3: Double agricultural productivity and income of smallholder farmers
- Target 2.c: Ensure proper functioning of food commodity markets

##  Key Features

- Real-time commodity price dashboard
- Price comparison across regional markets (Freetown, Bo, Kenema, Makeni)
- SMS and in-app price alert subscriptions
- Admin panel for market data officers
- Multi-language support (English and Krio)

##  Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **SMS Service:** Twilio API
- **Design:** Figma
- **Project Management:** GitHub, Agile Scrum
